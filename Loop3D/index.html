Loop3D Demos

Warning: The engine has been tested on Chrome, Firefox and Edge browsers on windows systems. Using a different system may lead to unexpected results.
Click a button to start a demo.
If performance is poor, ensure that the browser is running on a dedicated GPU.
Note: In the Tanks Demo, the red tank is moved with WASD keys, and the blue tank is controlled with arrow keys.
To shoot, use the spacebar for the red tank and the left mouse button for the blue tank.
